# Análisis Exploratorio de Anuncios de Venta de Coches

## Descripción
Esta aplicación web interactiva permite realizar un análisis exploratorio de datos de anuncios de venta de coches usados. Los usuarios pueden visualizar patrones y tendencias en los datos de manera fácil e intuitiva.

## Funcionalidades
- **Histograma interactivo**: Visualiza la distribución del kilometraje (odómetro) de los vehículos
- **Gráfico de dispersión**: Explora la relación entre el kilometraje y el precio de los coches
- **Interfaz intuitiva**: Controles simples con casillas de verificación y botones
- **Gráficos interactivos**: Visualizaciones dinámicas creadas con Plotly

## Datos utilizados
La aplicación analiza un conjunto de datos de anuncios de venta de coches (`vehicles_us.csv`) que incluye información sobre:
- Kilometraje (odómetro)
- Precios de venta
- Otros datos relevantes de vehículos

## Tecnologías utilizadas
- **Streamlit**: Framework para crear la aplicación web
- **Pandas**: Procesamiento y análisis de datos
- **Plotly**: Creación de gráficos interactivos
